## Overview
This repository showcases the professional capstone project **_Intimate Partner Violence (IPV) in Uganda: An Implementation-Ready Field Phase Model_**, developed as part of the Master of Public Health (MPH) degree at **The George Washington University Milken Institute School of Public Health**.  

Although academic in origin, the project is designed as a **field-deployable public-health initiative** integrating evidence-based project management, gender-based violence (GBV) ethics, and WHO operational frameworks.
